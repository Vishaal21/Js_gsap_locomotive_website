## Project Overview

This project showcases a modern, interactive website featuring animations and smooth scrolling, designed to enhance user experience.

## Features

- **Smooth Scrolling:** Powered by Locomotive Scroll for seamless navigation.
- **GSAP Animations:** Engaging animations for interactive elements.
- **Modern Aesthetics:** Clean and user-friendly design.

## Technologies Used

- **JavaScript**
- **GSAP (GreenSock Animation Platform)**
- **Locomotive Scroll**
